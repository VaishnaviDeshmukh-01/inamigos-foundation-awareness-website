# InAmigos Foundation - Awareness Website


An impactful and responsive website built to spread awareness about **InAmigos Foundation**, a Section 8 non-profit organization dedicated to creating positive change in education, environment, animal welfare, and community development.

---

## 🌟 About InAmigos Foundation

**InAmigos Foundation** was founded on **23rd September 2020** by **Mr. Govind Shukla** in Bilaspur, Chhattisgarh. 

We work towards building a better tomorrow through various initiatives:
- **Seva** - Community Service & Meals Distribution
- **Prakriti** - Environment & Tree Plantation
- **Bachpanshala** - Education for Underprivileged Children
- **Jeev** - Animal Welfare
- **Vikas** - Youth & Skill Development
- **Udaan** - Women Empowerment

---

## ✨ Features of this Website

- Fully Responsive Design
- Modern & Clean UI
- Interactive Navigation
- Impact Statistics
- Project Gallery
- Easy Donation Integration (Razorpay)
- Volunteer Registration
- SEO Friendly
